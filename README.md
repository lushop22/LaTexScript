# LaTexScript

Comandos a seguir:

## Clonar y crear carpeta
```
git clone https://github.com/lushop22/LaTexScript.git 
mv LatexScript ~/.LatexScript
```
## Exportar al path
En caso de usar zsh:
```
echo "export PATH=\$PATH:/opt/scripts" >> ~/.zshrc
```
En caso de usar bash:
```
echo "export PATH=\$PATH:/opt/scripts" >> ~/.bashrc
```
